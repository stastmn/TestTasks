# TestTasks

Функция печати двоичного представления числа работает следующим образом: в цикле на число итераций, равное числу бит в типе данных int, выполняется применение битовой маски с еденицей в самом старшем бите. По результату можно узнать состояние бита в самом старшем разряде числа. Далее выполняется битовый сдвиг влево и выполняется переход к новой итерации.
Название функции:  printToBinNumerals. Она находится в файле main.cpp

Для выполнения задания 2 помимо реализации функций  Serialize() и  Deserialize() пришлось написать вспомогательные функции Read() и Write(). 
Запись и чтение данных в бинарный файл я выполнил при помощи функций fwrite() и fread() соответственно. Сейчас я понял что не проверяю правильно ли произошла запись/чтение путем сравнения возвращаемого значения этих функций с колличеством байт, которые надо записать/считать. 
Запись в файл выполняется по следующему алгоритму: сначала в файл записывается количество узлов списка, далее для каждой строки из списка (строки хранятся в узлах списка) происходит запись размера строки (количество элементов) и собственно самой строки в виде массива символов char*.
По моей задумке, использование типа uint32_t для значений количества элементов позволит использовать функции сериализации и десериализации для машин с разным размером машинного слова.
Проверить запись в файл можно проверить при помощи любого hex редактора. Запись идет в файл file.bin (можно поменять в программе). Все функции и простая реализация двусвязного списка находятся в файле CustomList.h
